# Notion Clone

### Demo
https://bhagya-notion.vercel.app/

### Installation

Clone the repository
```bash

git clone https://github.com/allelbhagya/notion.git
cd notion-clone
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm start
```
